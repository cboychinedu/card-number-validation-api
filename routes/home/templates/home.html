💳 Card Validator API

Welcome! This API allows you to validate credit card numbers using the Luhn Algorithm.

Endpoint

POST /api

How to Use

For sending a valid card number:

Send a JSON request body containing the card number you wish to verify.

                {
                    "cardNumber": "401170277372404218"
                }

Example Response

                {
                    "status": "success",
                    "isValid": true,
                    "message": "Card number is valid!",
                    "cardNumber": "**************4218",
                    "statusCode": 200
                }

For sending an inValid card number:

Send a JSON request body containing the card number you wish to verify.

                {
                    "cardNumber": "401170277372404210"
                }

Example Response

                {
                    "status": "error",
                    "isValid": false,
                    "message": "Card number is not valid!",
                    "cardNumber": "**************4210",
                    "statusCode": 400
                }
